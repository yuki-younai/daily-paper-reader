## 摘要
多轮人机协作是部署自适应辅导、对话式推荐和专业咨询等交互式服务的基础。然而，通过强化学习优化这些交互受到可验证中间奖励的稀疏性以及用户响应高度随机性的阻碍。为了应对这些挑战，我们引入了隐式逐轮策略优化（ITPO）。ITPO 利用隐式过程奖励模型，从稀疏的结果信号中推导出细粒度的逐轮过程奖励。与波动较大的 token 级奖励不同，这些轮次级信号表现出卓越的鲁棒性，并可利用归一化机制进一步增强训练稳定性。我们在三个具有代表性的多轮协作任务（数学辅导、文档写作和医疗推荐）中评估了 ITPO。实验结果表明，当 ITPO 与 PPO、GRPO 或 RLOO 结合时，其收敛性始终优于现有基准。详尽的轨迹分析证实，ITPO 推断出的逐轮偏好在语义上与人类判断一致。代码已在 https://github.com/Graph-COM/ITPO 公开。

## Abstract
Multi-turn human-AI collaboration is fundamental to deploying interactive services such as adaptive tutoring, conversational recommendation, and professional consultation. However, optimizing these interactions via reinforcement learning is hindered by the sparsity of verifiable intermediate rewards and the high stochasticity of user responses. To address these challenges, we introduce Implicit Turn-wise Policy Optimization (ITPO). ITPO leverages an implicit process reward model to derive fine-grained, turn-wise process rewards from sparse outcome signals. Unlike volatile token-level rewards, these turn-level signals exhibit superior robustness and may utilize a normalization mechanism to further enhance training stability. We evaluate ITPO across three representative multi-turn collaborative tasks: math tutoring, document writing, and medical recommendation. Empirical results demonstrate that ITPO, when combined with PPO, GRPO, or RLOO, consistently achieves improved convergence than existing baselines. Elaborate trajectory analysis confirms that ITPO infers turn-wise preferences that are semantically aligned with human judgment. Code is publicly available at https://github.com/Graph-COM/ITPO.

论文聚焦于**多轮人机协作交互**（如数学辅导、医疗咨询）中的强化学习（RL）优化难题。核心挑战在于：
*   **奖励稀疏性**：通常只有对话结束时才有可验证的结果奖励（Outcome Reward），中间过程缺乏反馈。
*   **用户随机性**：用户响应具有高度不确定性，导致价值函数估计（Value Estimation）难以收敛。
*   **现有方法局限**：传统的 token 级隐式奖励模型（如 PRIME）虽然能提供细粒度反馈，但信号波动剧烈、缺乏语义解释性且容易过拟合。
**整体含义**：提出一种**逐轮（Turn-wise）**的隐式奖励分配机制，在保持自动化的同时，通过提升奖励信号的粒度和鲁棒性，解决多轮主动交互的对齐问题。

### 2. 方法论：核心思想与关键技术
ITPO 的核心是将 token 级的隐式过程奖励聚合为“轮次级”信号：
*   **隐式过程奖励模型（Implicit PRM）**：利用策略模型与参考模型之间的 token 级对数似然比（Log-likelihood Ratio）作为原始证据。
*   **轮次级聚合（Turn-wise Aggregation）**：将一个回合内所有 token 的隐式奖励求和，得到该轮次的原始贡献分数。
*   **归一化机制（Norm-ITPO）**：
    *   使用 **Softmax 函数**（含温度参数 $\eta$）对整条轨迹中各轮次的分数进行重新分配，计算出权重 $w_k$。
    *   将最终的结果奖励 $R$ 按权重 $w_k$ 分配给各个轮次，形成归一化的过程奖励。
*   **策略优化集成**：该奖励分配机制可无缝集成到 PPO（结合价值模型）、GRPO（组内标准化）或 RLOO（留一法基准）等主流优势函数估计器中。
*   **贝叶斯解释**：从理论上将该机制解释为对“关键轮次”的后验信念推断。

### 6. 主要结论与发现
*   **性能领先**：ITPO（尤其是 Norm-ITPO）在所有任务和算法组合中均显著优于稀疏奖励基线和 token 级 PRIME。
*   **稳定性高**：轮次级奖励比 token 级奖励收敛更快且波动更小，有效缓解了价值漂移问题。
*   **语义对齐**：ITPO 学习到的奖励分配与人类专家的判断高度一致（匹配率约 75%），能够准确识别出对话中的关键转折点。
*   **归一化的必要性**：实验证明，通过归一化校准奖励尺度，能显著提升 PPO 等依赖价值模型算法的训练稳定性。

### 7. 优点与亮点
*   **粒度适中**：巧妙地选择了“轮次”作为语义单元，既避免了 token 级的噪声，又解决了轨迹级的稀疏。
*   **无需额外标注**：完全依赖结果奖励进行隐式推导，解决了过程奖励模型（PRM）依赖昂贵人工标注的痛点。
*   **即插即用**：不改变现有 RL 算法的核心逻辑，仅作为一种奖励塑造（Reward Shaping）手段，易于推广。
*   **解释性强**：生成的轮次权重可直接用于分析模型在对话中的决策逻辑。

### 8. 不足与局限
*   **模拟器依赖**：实验高度依赖用户模拟器（User Simulator），模拟器与真实人类行为之间的分布差异（Distribution Shift）可能影响实际部署效果。

*   **计算开销**：虽然不需要人工标注，但在线 RL 过程中需要同时运行策略模型、参考模型和隐式 PRM，对显存和推理速度有一定要求。
