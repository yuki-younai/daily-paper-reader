## 摘要
大语言模型（LLMs）的飞速发展在广泛的任务中带来了显著的性能提升。然而，在处理超出模型上下文窗口限制的长文档时，无法通过单次处理完成整个上下文，因此需要进行分块处理。这需要通过多轮操作来读取不同的分块并更新记忆。然而，监督信号通常仅由最终结果提供，这使得在多轮训练设置中难以评估每一轮记忆更新的质量。这引入了时间信用分配（temporal credit assignment）的挑战。现有方法（如 LLM-as-a-judge 或过程奖励模型）会产生巨大的计算开销，并受到估计噪声的影响。为了更好地解决多轮记忆训练中的信用分配问题，我们提出了面向多轮强化学习的教师对齐奖励重塑（TAMTRL）。TAMTRL 将相关文档作为教师信号，通过将其与每一轮模型输入进行对齐，并以自监督的方式通过归一化概率分配奖励。这为每一次记忆更新提供了细粒度的学习信号，并提升了长上下文处理能力。在七个长上下文基准测试中对不同规模的多个模型进行的实验表明，TAMTRL 一贯优于强基准模型，证明了其有效性。我们的代码可在 https://anonymous.4open.science/r/TAMTRL-F1F8 获取。

## Abstract
The rapid progress of large language models (LLMs) has led to remarkable performance gains across a wide range of tasks. However, when handling long documents that exceed the model's context window limit, the entire context cannot be processed in a single pass, making chunk-wise processing necessary. This requires multiple turns to read different chunks and update memory. However, supervision is typically provided only by the final outcome, which makes it difficult to evaluate the quality of memory updates at each turn in the multi-turn training setting. This introduces a temporal credit assignment challenge. Existing approaches, such as LLM-as-a-judge or process reward models, incur substantial computational overhead and suffer from estimation noise. To better address the credit assignment problem in multi-turn memory training, we propose Teacher-Aligned Reward Reshaping for Multi-Turn Reinforcement Learning (TAMTRL). TAMTRL leverages relevant documents as teacher signals by aligning them with each turn of model input and assigns rewards through normalized probabilities in a self-supervised manner. This provides fine-grained learning signals for each memory update and improves long-context processing. Experiments with multiple models of varying scales across seven long-context benchmarks show that TAMTRL consistently outperforms strong baselines, demonstrating its effectiveness. Our code is available at https://anonymous.4open.science/r/TAMTRL-F1F8.

### 1. 核心问题与整体含义（研究动机）
*   **核心挑战**：LLM 处理超长文档时受限于上下文窗口，必须采用“分块读取+记忆更新”的多轮处理模式。
*   **信用分配难题（Credit Assignment）**：在强化学习（RL）训练中，通常只有最终答案能获得奖励（Outcome Reward）。如果最终答案错误，难以判断是哪一轮的记忆更新出了问题；反之，即使中间步骤很差，只要最后蒙对了也会获得奖励。
*   **现有方案不足**：使用“LLM作为裁判”或“过程奖励模型（PRM）”计算开销大，且容易引入外部模型的噪声。

### 2. 方法论：TAMTRL 框架
*   **核心思想**：利用**中心化训练、去中心化执行（CTDE）**的思路。在训练阶段，引入一个拥有“全局视角”的教师模型（即模型自身，但输入的是经过筛选的、仅含相关信息的文档分块）来指导只有“局部视角”的学生模型。
*   **关键技术细节**：
    1.  **教师对齐奖励重塑**：学生模型处理原始分块 $D_t$ 生成记忆 $M_{t+1}$；教师模型输入相关的过滤分块 $C_t$，计算学生生成的记忆在教师分布下的平均 Token 概率 $p_t$。
    2.  **归一化处理**：为了稳定训练，对概率得分进行长度归一化和全局 Min-Max 归一化，得到 $\hat{p}_t$。
    3.  **多轮奖励分配**：最终每轮的奖励 $R_t = \hat{p}_t \times \text{Outcome\_Reward}$。这意味着只有最终答案正确时，中间步骤才会根据与教师的对齐程度获得正向奖励。
    4.  **优化算法**：采用 DAPO（解耦剪切与动态采样策略优化）算法进行策略更新。
*   **理论支撑**：论文通过信息论分解证明了 TAMTRL 的目标函数可以拆解为：成功状态下的教师对齐项、失败状态下的正则项以及记忆与奖励之间的互信息项。

### 6. 主要结论与发现
*   **性能领先**：TAMTRL 在所有基准测试中均优于强基准模型，在 0.6B 和 1.7B 模型上分别实现了 1.87% 和 2.02% 的平均性能提升。
*   **泛化能力**：尽管训练时使用了带标注的相关文档作为教师信号，但模型在测试阶段处理无标注文档时依然表现出色。
*   **效率优势**：相比于依赖外部模型（LLM-judge）或额外训练模型（PRM）的方法，TAMTRL 在提升效果的同时显著降低了训练成本。

### 7. 优点（亮点）
*   **自监督信用分配**：无需外部昂贵的模型（如 GPT-4）作为裁判，利用模型自身的“局部-全局”视角差异产生监督信号。
*   **计算高效**：通过 Token 概率计算奖励，避免了昂贵的采样过程。
*   **理论完备**：提供了详尽的数学证明，解释了奖励重塑为何能促进模型学习。

### 8. 不足与局限
*   **依赖训练标注**：训练阶段需要知道哪些文档分块是“相关的”（用于构建教师输入），这在某些完全无监督的场景下可能难以获取。
*   **模型规模限制**：实验主要集中在 2B 以下的小规模模型上，尚未在大规模模型（如 70B+）上验证其扩展性。
*   **架构特定性**：该方法主要针对基于记忆机制（Memory-based）的长文本处理架构，对于其他长文本技术（如线性注意力或长 RoPE）的适用性未做讨论。
