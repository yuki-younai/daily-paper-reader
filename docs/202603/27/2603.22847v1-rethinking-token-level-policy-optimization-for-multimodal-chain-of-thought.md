## 摘要
多模态思维链（CoT）推理要求大型视觉语言模型构建将感知落地（perceptual grounding）与多步推理交织在一起的推理轨迹。然而，现有的带有可验证奖励的强化学习（RLVR）方法通常以粗粒度优化推理，统一对待 CoT，而没有区分它们不同程度的视觉落地。在这项工作中，我们对多模态推理轨迹进行了标记级（token-level）分析，并表明成功的推理具有结构化标记动态的特征，反映了感知落地和探索性推理。基于这一分析，我们提出了感知-探索策略优化（PEPO），它从隐藏状态相似性中推导出感知先验，并通过平滑门控机制将其与标记熵整合，从而产生标记级优势。PEPO 与现有的 RLVR 框架（如 GRPO 和 DAPO）无缝集成，既不需要额外的监督，也不需要辅助分支。在各种多模态基准测试上的广泛实验表明，与强大的强化学习基准相比，该方法在几何推理、视觉落地、视觉谜题解决和少样本分类等方面均取得了持续且稳健的改进，同时保持了稳定的训练动态。代码：https://github.com/xzxxntxdy/PEPO

## Abstract
Multimodal Chain-of-Thought (CoT) reasoning requires large vision-language models to construct reasoning trajectories that interleave perceptual grounding with multi-step inference. However, existing Reinforcement Learning with Verifiable Rewards (RLVR) methods typically optimize reasoning at a coarse granularity, treating CoT uniformly without distinguishing their varying degrees of visual grounding. In this work, we conduct a token-level analysis of multimodal reasoning trajectories and show that successful reasoning is characterized by structured token dynamics reflecting both perceptual grounding and exploratory inference. Building upon this analysis, we propose Perception-Exploration Policy Optimization (PEPO), which derives a perception prior from hidden state similarity and integrates it with token entropy through a smooth gating mechanism to produce token-level advantages. PEPO integrates seamlessly with existing RLVR frameworks such as GRPO and DAPO, requiring neither additional supervision nor auxiliary branches. Extensive experiments across diverse multimodal benchmarks demonstrate consistent and robust improvements over strong RL baselines, spanning geometry reasoning, visual grounding, visual puzzle solving, and few-shot classification, while maintaining stable training dynamics. #### 1. 论文的核心问题与整体含义
*   **研究背景**：大型视觉语言模型（LVLM）在处理复杂任务时，通常采用思维链（CoT）推理。目前主流的优化方法是使用“带有可验证奖励的强化学习”（RLVR），如 GRPO。
*   **核心问题**：现有的 RLVR 方法通常在**序列级别（Sequence-level）**进行优化，即对整个推理轨迹给予统一的奖励/优势值。这种“粗粒度”方法忽视了推理过程中不同标记（Token）的贡献差异：有些 Token 负责关键的视觉感知落地，而有些则负责逻辑探索。
*   **整体含义**：本文旨在通过**标记级别（Token-level）**的细粒度分析，识别并强化那些对感知和探索有重要贡献的 Token，从而提升多模态推理的准确性和一致性。

#### 2. 论文提出的方法论：PEPO
*   **核心思想**：提出“感知-探索策略优化”（Perception-Exploration Policy Optimization, PEPO），通过模型内部的隐藏状态相似度和输出熵，动态调整每个 Token 在策略梯度更新中的权重。
*   **关键技术细节**：
    1.  **感知建模（Perception Modeling）**：计算响应 Token 的隐藏状态与所有视觉 Token 隐藏状态之间的余弦相似度（VS）。相似度越高，表明该 Token 越依赖视觉信息（即“视觉落地”）。
    2.  **探索建模（Exploration Modeling）**：利用输出 Logits 计算 Token 熵（H）。高熵 Token 通常代表推理的决策点或不确定区域，需要更多探索。
    3.  **平滑门控融合（Smooth Gating Fusion）**：通过一个平滑门控机制将 VS 和 H 结合，生成 Token 权重 $w_t$。公式上利用 $tanh$ 激活函数进行中心化处理，确保感知信号占主导，同时由熵信号进行调节。
    4.  **Token 级优势分配**：将序列级的优势值 $A$ 重新分配为 Token 级优势 $A_t = [(1-\lambda) + \lambda w_t] A$，其中 $\lambda$ 随训练步数从 0 增加到 1，实现从序列级到 Token 级的平滑过渡。
