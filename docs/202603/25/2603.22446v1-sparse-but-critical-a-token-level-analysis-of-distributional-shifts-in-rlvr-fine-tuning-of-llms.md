## 摘要
具有可验证奖励的强化学习 (RLVR) 显著提升了大语言模型 (LLM) 的推理能力，但这些提升背后的词元级机制仍不清楚。我们对 RLVR 的分布效应进行了系统的实证研究，主要围绕三个核心分析展开：(1) 基础模型与 RL 模型之间分布偏移的词元级表征；(2) 通过交叉采样干预研究词元级分布偏移对序列级推理性能的影响；(3) 词元级偏移的细粒度机制。我们发现 RL 微调引发了高度稀疏且具有针对性的变化，只有极小比例的词元分布在基础策略和 RL 策略之间表现出显著差异。我们通过分析词元熵、位置集中度和概率质量的重新分配，进一步刻画了这些偏移的结构和演变。为了评估这些稀疏变化的功能重要性，我们进行了交叉采样实验，在不同的干预预算下，选择性地交换基础模型和 RL 模型之间的词元选择。我们表明，仅在基础模型的生成结果中插入极小比例的 RL 采样词元，就能逐步恢复 RL 的性能增益；而将同样少量的基础模型词元注入 RL 生成的序列中，则会导致性能坍缩至基础模型水平。这分离出了一组直接负责 RLVR 性能增益的极小词元级决策集。最后，我们探索了优势信号的散度加权变体作为诊断性干预，发现它们可以比基准线有所改进。总之，我们的研究结果揭示了 RLVR 引发的分布变化，并为将 RLVR 微调理解为一个有针对性的精炼过程提供了一个细粒度的词元级视角。

## Abstract
Reinforcement learning with verifiable rewards (RLVR) has significantly improved reasoning in large language models (LLMs), yet the token-level mechanisms underlying these improvements remain unclear. We present a systematic empirical study of RLVR's distributional effects organized around three main analyses: (1) token-level characterization of distributional shifts between base and RL models, (2) the impact of token-level distributional shifts on sequence-level reasoning performance through cross-sampling interventions, and (3) fine-grained mechanics of these shifts at the token level. We find that RL fine-tuning induces highly sparse and targeted changes, with only a small fraction of token distributions exhibiting meaningful divergence between the base and RL policies. We further characterize the structure and evolution of these shifts through analyses of token entropy, positional concentration, and reallocation of probability mass. To assess the functional importance of these sparse changes, we conduct cross-sampling experiments that selectively swap token choices between the base and RL models with varying intervention budgets. We show that inserting only a small fraction of RL-sampled tokens into base generations progressively recovers RL performance gains, while injecting a similarly small number of base token choices into otherwise RL-generated sequences collapses performance to base levels, isolating a small set of token-level decisions directly responsible for RLVR's performance gains. Finally, we explore divergence-weighted variants of the advantage signal as a diagnostic intervention, finding that they can yield improvements over baselines. Together, our results shed light on the distributional changes induced by RLVR and provide a fine-grained, token-level lens for understanding RLVR fine-tuning as a targeted refinement process.

#### 1. 论文的核心问题与整体含义
论文探讨了**带验证奖励的强化学习（RLVR）**（如 GRPO、DAPO 等）在提升大语言模型（LLM）推理能力时的底层机制。尽管 RLVR 在数学和逻辑竞赛（如 AIME）中取得了巨大成功，但学术界尚不清楚这种性能提升是源于模型行为的全局重写，还是局部关键决策的优化。研究动机在于通过**词元（Token）级别**的细粒度分析，揭示 RL 微调如何改变模型的预测分布，并确定哪些改变真正驱动了推理性能的飞跃。

#### 2. 论文提出的方法论
论文通过三个维度的分析构建了其方法论框架：
*   **分布偏移表征**：使用 **Jensen-Shannon (JS) 散度**来衡量基础模型（Base）与 RL 模型在相同上下文下的预测分布差异。选择 JS 散度而非 KL 散度是因为其对称性、有界性，且在处理 Top-p 截断分布时更加鲁棒。
*   **交叉采样干预（Cross-Sampling Interventions）**：
    *   **前向交叉采样**：在基础模型的生成过程中，当遇到高 JS 散度的位置时，强制改用 RL 模型的采样决策，观察性能恢复情况。
    *   **反向交叉采样**：在 RL 模型的生成过程中，将高散度位置的决策替换回基础模型的决策，观察性能坍缩情况。
*   **细粒度机制分析**：研究高散度位置的 Top-k 词元重叠度、词元排名变化以及概率质量的重新分配，探讨 RL 是在“创造新词”还是在“重新排序”。
*   **散度加权优势信号（Divergence-Weighted Advantage）**：作为一种诊断性尝试，根据 Token 级的 KL 散度大小对强化学习的优势信号（Advantage）进行加权（Sigmoid 变换），以引导模型关注特定类型的分布变化。

#### 3. 实验设计
*   **数据集/场景**：主要集中在数学推理任务，包括 **AIME 2024、AIME 2025、AMC** 以及研究生水平的科学问题集 **GPQA-Diamond**。
*   **模型与 Benchmark**：
    *   **基础模型**：Qwen2.5-32B、Qwen2.5-Math-7B、Qwen3-8B-Base、Mistral-Small-24B。
    *   **对比方法**：DAPO（解耦剪切与动态采样策略优化）、SimpleRL（基于 GRPO 的变体）、以及传统的监督微调（SFT）。
*   **评估指标**：Mean@32（32 次采样的平均准确率）、Pass@1、Token 级 JS 散度、词元熵（Entropy）等。

#### 5. 实验数量与充分性
*   **实验规模**：研究涵盖了从 7B 到 32B 的多个模型系列，对比了多种 RL 算法和 SFT，并在多个极具挑战性的数据集上进行了验证。
*   **充分性与客观性**：
    *   作者不仅分析了最终模型，还分析了训练过程中的中间 Checkpoints。
    *   通过改变 Top-p 采样参数和使用全量分布估计进行了鲁棒性检查。
    *   对比了 SFT 与 RL 的差异，证明了 RL 诱导的“稀疏性”并非微调的通用特性，而是 RL 特有的机制。
    *   实验设计严谨，通过前向和反向双向干预闭环验证了结论。

#### 6. 主要结论与发现
*   **分布偏移极其稀疏**：RLVR 引起的变化具有高度局部性。在 SimpleRL 中，超过 98% 的位置分布几乎没有变化；在 DAPO 中这一比例也超过 80%。
*   **稀疏点决定性能**：交叉采样实验显示，仅修改不到 5% 的关键 Token 决策，就能让基础模型获得 RL 级别的性能，或让 RL 模型性能彻底坍缩。
*   **位置集中效应**：分布偏移主要集中在序列的**开头**（决定推理路径）和**结尾**（格式化答案），中间过程相对稳定。
*   **重新排序而非创造**：在高散度位置，RL 模型很少提升原本概率极低的词元，而是对基础模型已经认为“合理”的 Top-k 候选词进行重新排序和概率放大。
*   **与 SFT 的本质区别**：SFT 引起的分布变化更广泛、更密集，而 RL 像一把“手术刀”，只针对影响奖励的关键决策点进行精炼。

#### 7. 优点
*   **视角新颖**：从 Token 级分布动力学角度解释了 RLVR 的成功，填补了宏观性能指标与微调机制之间的鸿沟。
*   **干预实验极具说服力**：通过交叉采样直接建立了“局部分布变化”与“全局推理成功”之间的因果关系。
*   **细粒度洞察**：揭示了 RL 主要是通过“轨迹引导（Steering）”而非“知识灌输”来提升推理能力的本质。

#### 8. 不足与局限
*   **任务局限性**：研究主要集中在具有**可验证奖励（Verifiable Rewards）**的推理任务（数学、代码、逻辑）。在主观性较强、缺乏明确奖励函数的通用对话或创意写作任务中，这种稀疏性结论是否成立尚待验证。
*   **诊断性实验尚浅**：散度加权优势信号的尝试虽然显示出潜力，但仍处于初步阶段，尚未形成一套通用的、能显著超越现有 SOTA 的新算法。
*   **计算成本高昂**：由于需要获取完整的词元分布概率，此类分析在超大规模模型上的复现成本极高。
